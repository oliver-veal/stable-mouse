## Getting Started

To get a local copy up and running follow these steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/oliver-veal/stable-mouse.git
   ```
2. Serve src/index.html (I recommend live-server or http-server from npm)
